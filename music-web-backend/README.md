# Adonis fullstack application

Musical Web 

## Setup

```js
run `npm install`.
```

### Migrations

tạo database rồi khởi tạo bảng
```js
adonis migration:run
```

### Start
tạo file `.env` giống với file `.env.example`, tạo database `music_web` trong mySQL, điền mật khẩu root vào file .env
```
adonis serve --dev
```
